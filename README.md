## Company Website

This project is a company website developed to support information delivery and internal digital needs.  
The website provides structured company information, service details, and interactive features to improve user experience and accessibility.

The system was developed end-to-end, covering planning, UI design, front-end implementation, back-end development, and database integration.  
It focuses on responsive design, clean code structure, and maintainable architecture.

### Features
- Company profile and service information pages  
- Responsive and user-friendly interface  
- Dynamic content management  
- Database integration for data processing  
- Website performance optimization and maintenance  

#### Tech Stack
- Front-End: HTML, CSS, JavaScript, Bootstrap  
- Back-End: PHP / CodeIgniter 3 / Node.js (adjust as needed)  
- Database: MySQL
